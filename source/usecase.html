Typical Use Cases¶

This library, in its current state, can be used for acceleration of data compression applications in two ways:

Acceleration Scope

Developer’s Usage of Compression Library

Individual Components

Write a custom kernel with modules from the library.

Compression-Decompression

Use a complete compression or decompression kernel (eg. lz4, snappy)

L1 module contains several primitive components which can be used in different algorithm kernels. Developers who only need these primitives to build their own kernels may refer L1 Module User Guide.

L2 module contains pre-designed compression kernels for LZ4 and Snappy compression and decompression. Developers can directly use these kernels in their design and may refer L2 Kernel User Guide.
