    uint32_t max_lit_limit [PARALLEL_BLOCK],
    uint32_t input_size,
    uint32_t core_idx

Lz4 compression engine.

Parameters:

MIN_OFFSET

lowest match distance

minimum match length

LZ_MAX_OFFSET_LIMIT

OFFSET_WINDOW

BOOSTER_OFFSET_WINDOW

LZ_DICT_SIZE

MAX_MATCH_LEN

MATCH_LEN

match length

MATCH_LEVEL

number of levels to check for match

inStream

input hls stream

lz4Out

output hls stream

lz4Out_eos

output end of stream indicator

lz4OutSize

output compressed size

max_lit_limit

maximum literals before match

input_size

input data size

core_idx

engine index
