.. rubric:: Parameters:

.. list-table::
    :widths: 20 80

    *
        - typeIn

        - 0: Cap, 1: Floor. For Collar mode, it is actually the result of Cap mode minus the result of Floor mode.

    *
        - forwardRateIn

        - base of forward rate

    *
        - cfRateIn

        - cfRateIn[0]: Cap strike rate, cfRateIn[1]: Floor strike rate

    *
        - nominalIn

        - nominal principal

    *
        - gearingIn

        - gearing

    *
        - accrualTimeIn

        - accrual time

    *
        - size

        - the actual size of array timeIn

    *
        - timeIn

        - array time, the difference between the maturity date and the reference date, unit is year.

    *
        - rateIn

        - array rate
